Developer Express Example
How to send authenticated requests to the OData service

This example demonstrates how to send authenticated requests to the OData service. The main idea is to handle the ODataContext.beforeSend (http://js.devexpress.com/Documentation/ApiReference/Data_Library/ODataContext/Configuration/#beforeSend) event and add login information to the header of your request. The http://www.devexpress.com/scid=E4403 example demonstrates how to process requests on the server side.

You can find sample updates and versions for different programming languages here:
http://www.devexpress.com/example=E4460.
